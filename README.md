# Drea — Andrea's Personal Fitness Companion

A personal fitness PWA built exclusively for Andrea. Combines a guided workout flow with progressive overload suggestions, workout history and progression tracking, personal record detection, body weight tracking, and an AI coaching layer powered by Claude.

## Features
- Split-based workout selection with exercise frequency sorting
- Per-set logging with smart progressive overload suggestions
- Workout history with progression charts
- Personal record detection and celebration
- Body weight tracking with trend visualization
- AI coach powered by Claude

## Tech Stack
- Single-file HTML architecture (React 18 via CDN, Tailwind CSS via CDN)
- Firebase Realtime Database for persistent storage
- PWA with service worker for offline support
- Deployed on Netlify
